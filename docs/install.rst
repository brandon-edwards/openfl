********************
Install the Software
********************

Depending on how you want to set up your federation, choose one of the following installation procedure.

:doc:`install.initial`

    - prepares the environment and installs the |productName| package.

:doc:`install.docker`

    - builds a Docker\*\ image of |productName|, which you can use use to run your federation in an isolated environment.
